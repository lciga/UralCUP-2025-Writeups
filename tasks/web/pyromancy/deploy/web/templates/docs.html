Getting Started with Pyromancy

Pyromancy is a Pyro4 and pickle-based server for managing your python handlers.

Note: Handlers in context of Pyromancy - is a little python applications that can run and interact with eachother using functionality, provided by Pyro4 and our infrastructure.

How to start?

  • You can get your free temporary account using CreateFreeUser() on our publicly available Pyromancy repository

Functions

In order to interact with Pyromancy server you should use Pyro4 client. You should write your own or use toolkit provided by Pyro4.

Publicly available functions

Here is a list of available functions:

# Returns information about Pyromancy endpoint PrintInfo() # Creates free user and returns it's token CreateFreeUser() # Puts your b64encoded pickle to our server PutObject(unique_token:string, object:b64string) # Returns your objects GetObjectList(unique_token:string)
